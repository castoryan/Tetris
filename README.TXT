PROJECT TITLE: Tetris
PURPOSE OF PROJECT: The Final project of OOP course, Fall 2013-2014 , KU Leuven-GroupT
VERSION or DATE: 12/12/2013
HOW TO START THIS PROJECT:

For the Game Class, just build a instance of it.
For the Game2 Class, We built two Constructors, 
The first one was setted as 12*12 game board,and speed at 1000ms.
And the second Constructor, players can set the game speed and the size of board whatever they like.

AUTHORS: Qinrui Yan & Shiying Yu

USER INSTRUCTIONS:

This game is totally controlled by keyboard. 

Button UP:      rotate the shape;
Button LEFT:    move left the shape;
Button RIGHT:   move right the shape;
Button DOWN:    speed up moving down the shape;
